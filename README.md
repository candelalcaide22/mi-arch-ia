# 🏗️ ARCH-IA 1.0
### Conversor Inteligente de Planos 2D a 3D

**ARCH-IA** es una herramienta diseñada para arquitectos que permite transformar coordenadas de planos en estructuras 3D sólidas de forma automática.

## 🚀 ¡Úsala ahora mismo!
Haz clic en el siguiente enlace para acceder a la aplicación web:
[ENTRAR EN ARCH-IA](AQUÍ_PEGA_TU_LINK_DE_STREAMLIT)

## 🛠️ Cómo funciona
1. Sube tu archivo `.xyz`.
2. Elige la altura de los muros.
3. Descarga el archivo `.dxf` listo para AutoCAD.
